# TaskReview

O **TaskReview** é um sistema de gerenciamento de tarefas (checklist) simples e intuitivo, desenvolvido para ajudar na organização do dia a dia. Com uma interface limpa, ele permite adicionar, marcar e remover tarefas de forma rápida.

---

## Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout responsivo e transições.
* **JavaScript (Vanilla):** Lógica de manipulação do DOM.

---

## Funcionalidades

* [x] **Adicionar Tarefas:** Criação rápida de novos itens na lista.
* [x] **Concluir Tarefas:** Marque tarefas finalizadas com um simples clique.
* [x] **Remover Tarefas:** Exclua itens da lista individualmente.
* [x] **Responsividade:** Interface adaptada para dispositivos móveis e desktop.
